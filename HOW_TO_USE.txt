DISCORD RESOLVER (VERY EASY)

--------------------------------
JUST DO THIS
--------------------------------

1. Request your Discord data
   (Settings → Privacy → Request Data)

IMPORTANT:
Make sure your data includes:
- Account
- Messages
- Servers

2. Download the ZIP Discord gives you

3. EXTRACT the ZIP

4. Put ALL folders into ONE folder

Example:
DISCORD/
 ├── Account
 ├── Messages
 ├── Servers

5. Drag that folder onto:
   resolver.exe

DONE.

--------------------------------
WHERE ARE MY FILES?
--------------------------------

After it finishes, you will see:

DISCORD_resolved
DISCORD_readable

They appear NEXT TO your Discord folder.

--------------------------------
OPEN YOUR CHATS
--------------------------------

Open:
*_readable

Then click the HTML files.

--------------------------------
TOKEN (OPTIONAL BUT BETTER)
--------------------------------

WHAT IT DOES:
The token lets the tool get usernames from Discord.
Without it, some IDs may stay as numbers.

--------------------------------
OPTION 1: BOT TOKEN (RECOMMENDED)
--------------------------------

1. Go to:
https://discord.com/developers/applications

2. Click "New Application"

3. Go to:
Bot → Add Bot

4. Click:
"Reset Token" → Copy the token

5. Put it into a file called:
token.txt

--------------------------------
OPTION 2: USER TOKEN (ADVANCED)
--------------------------------

1. Open Discord in your browser
2. Press F12
3. Go to "Network"
4. Refresh the page
5. Click any request
6. Find "authorization"
7. Copy the long text

Paste into token.txt

--------------------------------
IMPORTANT
--------------------------------

- NEVER share your token
- Bot token is safer than user token

--------------------------------
HIDE YOURSELF (OPTIONAL)
--------------------------------

1. Make a file called:
ignore_ids.txt

2. Put your ID inside:
123456789012345678

3. Put it next to resolver.exe

--------------------------------
FINAL SETUP
--------------------------------

resolver.exe
ignore_ids.txt (optional)
token.txt (optional)

--------------------------------
IF IT DOESN’T WORK
--------------------------------

You probably:
- didn’t extract the ZIP
- missing required folders
- put files in wrong place

Fix that and try again.

--------------------------------
DONE
--------------------------------

That’s it.
